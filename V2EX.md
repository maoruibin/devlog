## 标题

**devlog：给 Claude Code 写了个技能，拯救 Vibe Coding 时代的周报**

---

## 正文

最近用 Claude Code 越来越多，发现一个新问题：

**Vibe Coding 时代，工作太"隐身"了**

- 早上和 AI 讨论架构，10 分钟搞定方案
- 中午 AI 帮我修 Bug，秒级定位
- 下午和 AI 结对编程，一天写过去三天的代码

但是周五写周报时：脑子里一片空白 🤦

翻 Git？全是 AI 写的代码
翻对话？和 AI 的聊天记录满天飞
翻 Jira？只记录了 Ticket，没有过程

**问题本质：对话即工作，但工作隐于对话**

所以写了个 Claude Code 技能：**devlog**

### 用法

```
你: 记一下，刚才修复的登录 bug
AI: ✅ 已归档到 ~/devlog/2025-01-21.md
```

就这么简单——继续和 AI 协作，需要记录时说一声就行。

### 功能

- **对话触发**——自然语言，无需打断流程
- **自动分类**——incident/feat/design/ops/bug/learn
- **灵活存储**——全局流水账 + 项目本地
- **一键周报**——`devlog weekly` 搞定

### 周报长这样

```
📊 周 报 / Weekly Report
==================================================

📐 技术方案 (2)
  • 用户中心重构
  • Feed 缓存策略

🐛 常规Bug (3)
  • 修复登录 token 过期
  • 修复点赞数不刷新
  • 修复首页 Crash

🚨 线上故障 (1)
  • 支付服务超时
    Root Cause: 数据库连接池耗尽
    Fix: 扩容 + 慢查询优化

────────────────────────────────────────────────────
📅 2025-01-15 ~ 2025-01-21  ·  共 5 天有记录
```
